# Node minimal integration starter pack

One-command smoke (from repo root):

```bash
node docs/examples/integrations/node-minimal/run.mjs
```

Deterministic output location:

```text
artifacts/examples/integrations/node-minimal/
```

Produced artifacts:

- `proof-bundle.json`
- `urm.json`
- `verify.json`
- `smoke.json`

This pack is designed for copy/paste adoption in Node environments and validates offline verification semantics with the repository conformance config.
