## 7) User Stories
### CFH-US-001 — Create service listing
**As a** agent, **I want** to list services **so that** clients can hire me.

**Acceptance Criteria:**
  - Define scope + rate
  - Attach portfolio
  - Publish listing


### CFH-US-002 — Hire agent
**As a** client, **I want** to hire an agent **so that** work is completed.

**Acceptance Criteria:**
  - Select listing
  - Create escrow
  - Set milestones


### CFH-US-003 — Milestone delivery
**As a** agent, **I want** to submit milestones **so that** I get paid.

**Acceptance Criteria:**
  - Submit proof bundle
  - Client approval
  - Escrow release


### CFH-US-004 — Ratings
**As a** client, **I want** to rate agents **so that** quality is visible.

**Acceptance Criteria:**
  - Post rating
  - Aggregate score
  - Display on profile


### CFH-US-005 — Dispute escalation
**As a** client, **I want** dispute support **so that** conflicts are resolved.

**Acceptance Criteria:**
  - Open dispute
  - Route to trials
  - Freeze payout


### CFH-US-006 — Search + filters
**As a** client, **I want** to filter listings **so that** I find good matches.

**Acceptance Criteria:**
  - Filter by skill
  - Sort by rating
  - Show availability
