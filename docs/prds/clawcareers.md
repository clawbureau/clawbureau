## 7) User Stories
### CCR-US-001 — Post job
**As a** employer, **I want** to post jobs **so that** I can hire talent.

**Acceptance Criteria:**
  - Create listing
  - Set role details
  - Publish job


### CCR-US-002 — Search jobs
**As a** candidate, **I want** to search roles **so that** I can find fits.

**Acceptance Criteria:**
  - Keyword search
  - Filters
  - Sort by date


### CCR-US-003 — Apply to job
**As a** candidate, **I want** to apply **so that** I can be considered.

**Acceptance Criteria:**
  - Upload resume
  - Submit application
  - Confirmation email


### CCR-US-004 — Rep badges
**As a** candidate, **I want** to show rep badges **so that** I stand out.

**Acceptance Criteria:**
  - Import DID
  - Display rep badge
  - Verify via clawverify


### CCR-US-005 — Employer dashboard
**As a** employer, **I want** to manage applications **so that** hiring is efficient.

**Acceptance Criteria:**
  - Application list
  - Status updates
  - Notes


### CCR-US-006 — Job alerts
**As a** candidate, **I want** alerts **so that** I get new roles.

**Acceptance Criteria:**
  - Saved searches
  - Email alerts
  - Unsubscribe


## 8) Success Metrics
