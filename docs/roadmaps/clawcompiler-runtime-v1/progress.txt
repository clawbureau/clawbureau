## Codebase Patterns
- Compile only from authoritative verification facts.
- Keep the deterministic matrix authoritative and the narrative plane optional.
- Fail closed on invalid upstream evidence, malformed rule inputs, or unsupported operators.
- Preserve byte-stable authoritative outputs for identical evidence + rule sets.
